# Battle-Frontier
## Author
Deepak Ramalingam
## Building Server + WebApp
```sh
npm install
```
## Running Server + WebApp in Development
```sh
npm run start --prefix WebApp
```
```sh
npm run start --prefix Server
```
## Running Server + WebApp in Production
```sh
npm run start --prefix Server
```
## Running on iOS
```sh
cd MobileApp
npm install
npx react-native run-ios
```
## Running on Android
```sh
cd MobileApp
npm install
npx react-native run-android
```
